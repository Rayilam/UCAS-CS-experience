### 世界文学与写作(2019 spring)
上课和作业很水。其中有4次小测。都是一中午能复习完的类型。上课有听基本没问题。  
小测有ABC卷，期末考会有小测原题。  
一篇作文，Each student should compose an essay comparing/ or contrasting 2 of the literature pieces that we have read. 写完期末考的最后一题大题可直接用上。
